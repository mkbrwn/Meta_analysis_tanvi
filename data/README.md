# Data Directory

This folder stores raw and intermediate data files used for the CAP meta-analysis workflow.

## Contents

- `CAP_mortality.xlsx`: Source extraction workbook for CAP mortality-related study data.
- `icu_cardiac_arrest_extracted.xlsx`: Source extraction workbook for ICU cardiac arrest-related data.
- `data_extraction_tanvi_050626/`: Versioned extraction snapshot folder.
  - `merged_all_sheets.csv`: Merged output across extraction sheets.
